## Cara Install
<ol>
    <li> Buat database
    <li> Ubah file .env.example menjadi .env, buka lalu isi DB_DATABASE, DB_USERNAME, dan DB_PASSWORD (jika ada). 
    <li> buka cmd lalu arahkan ke file, kemudian tulis "php artisan migrate" (tanpa kutip) lalu enter.
    <li> selesai.
